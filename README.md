# CheeseQuiz Post Bot

Telegram-бот для публикации постов в канал [@CheeseQuizBg](https://t.me/CheeseQuizBg) с кнопкой записи на игру.

## Возможности

- Создание поста: фото + текст + inline-кнопка
- Предпросмотр перед публикацией
- Редактирование фото/текста до отправки
- Ограничение доступа по admin ID

## Установка

```bash
cd quiz_post_bot
pip install -r requirements.txt
```

## Настройка

1. Создай `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

2. Заполни переменные:
   - `BOT_TOKEN` — токен от [@BotFather](https://t.me/BotFather)
   - `ADMIN_IDS` — Telegram user ID админов через запятую
   - Остальные параметры уже заполнены по умолчанию

3. Бот должен быть **администратором** канала `@CheeseQuizBg` с правом отправки сообщений.

## Запуск

```bash
python bot.py
```

## Использование

1. `/newpost` — начать создание поста
2. Отправить фото
3. Отправить текст (поддерживается HTML-разметка)
4. В предпросмотре: опубликовать, изменить фото/текст или отменить
